# 교육용 근전도 판독 보조 웹앱

학생 교육용 규칙 기반 근전도/신경전도 판독을 위한 보조 도구입니다.  
실제 임상 진단을 대체하지 않으며, 학습용 분류 연습과 해부학적 분포 이해를 돕기 위한 목적입니다.

---

## 주요 특징

- 사례 학습을 통해 대표적인 사례별 검사결과 및 해석 이해
- 체크형 입력과 수치형 입력 중 선택 가능
- 정상쪽 / 병변쪽 비교 개념 반영
- 팔/다리/척추주위근 중심의 교육용 구조
- 모바일/태블릿/PC에서 사용하기 쉬운 탭형 구조
- 자동 결과 출력:
  - 최종 유력 진단
  - 손상 의심 신경
  - 신경학적 레벨/분절
  - Top 3 감별진단
  - 병변 수준/패턴
  - 추가 검사 권고
  - 이상 항목 상세
